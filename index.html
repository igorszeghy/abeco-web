Demo 1

Faktur-bot — automatické spracovanie faktúr

Faktúra príde na email → AI ju prečíta a extrahuje všetky údaje → automaticky zapíše do registra a ERP systému. Ľudský zásah len pri výnimkách.

Napojenie na: Google Sheets Pohoda REST API Money S3 Abra Flexi KROS Omega Excel / OneDrive Slack / Teams notif.

💡 Ukážka beží na Google Sheets. Pre váš ERP nastavíme priame API napojenie — dáta nikdy neopustia váš systém.

🔍 zväčšiť
1 Faktúry prichádzajú emailom — rôzni dodávatelia, rôzne formáty PDF aj Word. Systém rozpozná všetky automaticky.
🔍 zväčšiť
2 AI extrahuje dodávateľa, IČO, IČ DPH, číslo a dátum faktúry — zapíše do Google Sheets, Pohody alebo iného ERP
🔍 zväčšiť
3 Porovnanie: originálna faktúra PDF verzus vyplnený register — presné údaje bez ručného prepisovania
🔍 zväčšiť
4 Dashboard v reálnom čase: počet faktúr, úspešnosť, celková suma, faktúry po splatnosti — pre manažment aj účtovníka
🔍 zväčšiť
5 Make.com workflow pod kapotou: Gmail sleduje poštu → CloudConvert konvertuje PDF → OpenAI Vision číta faktúru → JSON parsuje dáta → Google Sheets / Pohoda / ERP zapíše riadok
